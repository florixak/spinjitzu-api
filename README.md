# 🥷 Spinjitzu API

> A modern, well-documented REST API providing structured data from the Ninjago universe.

Spinjitzu API is a personal backend project built to practice modern backend development while creating a public API inspired by projects such as SWAPI. The goal is to provide clean, consistent, and easy-to-consume endpoints for developers interested in building applications around the Ninjago universe.

Rather than focusing only on exposing data, the project emphasizes software architecture, maintainability, security, and production-ready development practices.

## Goals

- Build a high-quality public REST API
- Follow clean architecture and REST principles
- Practice NestJS and modern backend development
- Provide comprehensive API documentation with Swagger
- Ensure consistent responses and robust validation
- Keep the project secure, scalable, and easy to maintain

## Tech Stack

- **Framework:** NestJS
- **Language:** TypeScript
- **Database:** PostgreSQL
- **ORM:** Drizzle ORM
- **Authentication:** JWT
- **Documentation:** Swagger (OpenAPI)
- **Containerization:** Docker
- **Testing:** Jest

---

> This project is intended as both a learning experience and a production-quality portfolio project, following modern backend engineering practices from development to deployment.
